## 🚀 Deployment

### AWS S3

1. Create S3 bucket with static website hosting enabled
1. Make bucket public with bucket policy
1. Upload `index.html` and `photo.jpeg`
1. Access via S3 website URL

### Netlify (Free Forever)

1. Connect GitHub repository to Netlify
1. Deploy with default settings
1. Add custom domain `chinonsocloudsec.online`
1. Update nameservers on Namecheap
